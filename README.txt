// Nama Kelompok : Gorengans

// Project : counterCraft
Game seperti counterStrike dengan character dan map seperti mineCraft.
Terdapat 2 player yg dapat dimainkan.

Player akan menang jika berhasil mengenai Player lain sebanyak 5 kali,
dan akan reload peluru selama 3 detik setelah menembak sebanyak 20 kali.
